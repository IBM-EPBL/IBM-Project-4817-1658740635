2.Data Preparations:

Some prevalent challenges faced while collecting data are inconsistent data, ambiguous data, duplicate data, inaccurate data, too much data, etc.

To overcome this problem data preparation is done.

It is the process of profiling, cleansing, transforming and validating data.

  • Data structuring

3.Implementation Details:

Data analytics for analysing and estimating traffic and capacity on major ports is done by creating various graphs and charts

Plotting different graphs give broad understanding about the data and relationship among the features of data.

This supports in creating meaningful dashboards for exploring the data and making future predictions from them.
